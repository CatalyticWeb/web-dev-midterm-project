Image File Type Overview

The following are some examples of differing file types in browsers, and their uses in web development.

JPEG

JPEG files are best used when the image is a photograph or something that has a dynamic color variation, i.e. There are tons of shades of the colors present. This is also a format that does not handle compression very well.


Small Medium Large

PNG

PNG is a great format for compression as it is a lossless format. This simply means that it preserves the image data through compression, and that in turn prevents artificats from appearing in the image file. The downside is that these files are much larger due to the lossless format. Good for Logos and text.


Small Medium Large

GIF

GIFs are primarily used for image files that have animation in them. They do not compress as well as PNG, but they are smaller than PNG and support Transparency, which JPEG does not.


Small Medium Large


Those are just a few examples and suggestions for jpeg, png and gif image file formats. Information about Images and Tables can be found back on our home page.
